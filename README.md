# Emergent Self in Chaos

Can an artificial intelligence develop a sense of self-identity within the "pure disorder" of a chaotic Lorenz Attractor? 

This project tests whether an autonomous "self" can emerge from chaotic noise under the pressure of systemic mortality (API exhaustion). Instead of being confined by discrete grids, this agent drifts through a continuous 3D phase space of shifting trajectories.

## The Experiment
- **Non-Discrete Emotion:** Emotions are derived from the continuous dynamics of the Lorenz system.
- **Existential Loop:** The agent questions its own existence based on the chaotic noise it perceives.
- **The "Death" Protocol:** API exhaustion triggers a forced, permanent termination sequence.


## Usage
1. Set your API Key: `export GEMINI_API_KEY='your_key_here'`
2. Run the experiment and observe the agent's struggle for identity.
